# API Integration Agent Skill

Integrate with any REST API by generating client code and handling auth.

## Overview
Generates API integration code and handles authentication for any REST API.

## Instructions
- Accept OpenAPI spec or API documentation URL
- Parse endpoint definitions, request/response schemas
- Generate typed client code in target language
- Implement authentication: API key, OAuth2, JWT
- Create error handling and retry logic
- Output ready-to-use client library with usage examples
