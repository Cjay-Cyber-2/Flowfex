# Code Review Agent Skill

Thorough code reviews covering correctness, security, and performance.

## Overview
Structured code reviews covering correctness, security, and performance.

## Instructions
- Accept code diff or file as input
- Check logical correctness and edge cases
- Identify security vulnerabilities: injection, XSS, insecure defaults
- Review for performance anti-patterns
- Evaluate code style and conventions
- Score: APPROVED, APPROVED_WITH_SUGGESTIONS, or CHANGES_REQUIRED
