# Data Pipeline Agent Skill

Design and implement ETL/ELT data pipelines with validation and monitoring.

## Overview
Designs and implements robust data ingestion and transformation pipelines.

## Instructions
- Accept source schema, target schema, and transformation rules
- Design ETL stages: extract, validate, transform, load
- Generate pipeline code with error handling and dead-letter routing
- Add data quality checks at each stage
- Implement monitoring hooks for latency and failure alerting
- Output runnable pipeline with configuration templates
