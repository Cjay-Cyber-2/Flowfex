# DevOps Automation Agent Skill

Automate CI/CD pipelines, infrastructure provisioning, and deployments.

## Overview
Automates DevOps workflows from CI/CD pipelines to infrastructure provisioning.

## Instructions
- Accept deployment target and application type
- Generate CI/CD pipeline configuration (GitHub Actions, GitLab CI)
- Create infrastructure-as-code templates (Terraform, Pulumi)
- Design blue-green or rolling deployment strategy
- Add health checks and automatic rollback triggers
- Output complete DevOps configuration package
