# Documentation Writer Agent Skill

Auto-generate API docs, READMEs, and technical guides from source code.

## Overview
Generates comprehensive technical documentation from code and specs.

## Instructions
- Accept source code, API spec, or feature descriptions
- Extract public interfaces, functions, and parameters
- Generate README with installation, usage, and examples
- Write API reference docs with request/response examples
- Create architecture decision records (ADRs)
- Follow docs-as-code principles with versioning support
