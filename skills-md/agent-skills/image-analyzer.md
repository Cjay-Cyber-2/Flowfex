# Image Analysis Agent Skill

Analyze images for objects, text, sentiment, quality, and actionable insights.

## Overview
Comprehensive image analysis covering objects, text, quality, and insights.

## Instructions
- Accept image file or URL as input
- Identify objects, people, scenes, and activities
- Extract and recognize text in the image (OCR)
- Assess image quality: sharpness, exposure, composition
- Detect sentiment from facial expressions and body language
- Output structured analysis with confidence scores per category
