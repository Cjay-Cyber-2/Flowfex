# Prompt Optimizer Agent Skill

Optimize LLM prompts for clarity, specificity, and output quality.

## Overview
Analyzes and rewrites prompts to improve output quality and reduce failures.

## Instructions
- Accept original prompt
- Analyze for vagueness, missing constraints, implicit assumptions
- Apply best practices: chain-of-thought, role definition, output format
- Generate improved prompt with explicit structure
- Test improved prompt on sample inputs
- Output side-by-side comparison with rationale
