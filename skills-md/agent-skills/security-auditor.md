# Security Auditor Agent Skill

Audit codebases and infrastructure for security vulnerabilities.

## Overview
Systematic security auditing of code and infrastructure configurations.

## Instructions
- Scan codebase for common vulnerabilities: OWASP Top 10
- Check for hardcoded secrets, weak cryptography, injection risks
- Audit dependency versions against CVE databases
- Review infrastructure config for misconfigurations
- Generate severity-ranked findings with remediation steps
- Output a security report in SARIF or markdown format
