# SQL Agent Skill

Translate natural language questions into optimized SQL queries.

## Overview
Converts natural language questions into correct, optimized SQL queries.

## Instructions
- Accept database schema and user question
- Parse relevant tables and relationships from schema
- Generate syntactically correct SQL for the question
- Optimize query: add appropriate indexes, avoid N+1
- Explain each clause in plain English
- Validate query safety: no destructive operations without confirmation
