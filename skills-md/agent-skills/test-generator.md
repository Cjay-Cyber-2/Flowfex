# Test Generator Agent Skill

Generate comprehensive unit, integration, and E2E tests from source code.

## Overview
Automatically generates test suites from production code and specifications.

## Instructions
- Accept source code or function signatures as input
- Analyze code paths and branch conditions
- Generate unit tests covering happy path, edge cases, and error cases
- Create integration tests for API endpoints and database operations
- Write E2E tests for critical user journeys
- Output test files with descriptive test names and assertions
