# Translation Agent Skill

Translate content between languages with cultural adaptation and terminology consistency.

## Overview
High-quality language translation with cultural adaptation and consistency enforcement.

## Instructions
- Accept source text and target language(s)
- Apply domain-specific glossary for terminology consistency
- Translate with cultural adaptation, not just literal mapping
- Maintain formatting: markdown, HTML tags, code blocks
- Flag culturally sensitive content for human review
- Output translation with confidence score and notes
