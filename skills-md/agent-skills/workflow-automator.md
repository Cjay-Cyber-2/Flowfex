# Workflow Automator Agent Skill

Design and execute multi-step automation workflows across any tool stack.

## Overview
Designs and executes complex multi-step automation workflows.

## Instructions
- Accept workflow goal in natural language
- Decompose into atomic, sequenced steps
- Map each step to available tools and integrations
- Handle branching logic and conditional steps
- Execute workflow with full state tracking
- Report completion status and any exceptions with recovery options
