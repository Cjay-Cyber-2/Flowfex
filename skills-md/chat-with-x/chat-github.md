# Chat with GitHub

Ask questions about any GitHub repository: code, issues, and documentation.

## Overview
Natural language interface to GitHub repositories.

## Instructions
- Access target GitHub repository
- Index code files, README, issues, and PR comments
- Chunk with file path context preserved
- Answer questions about codebase or project status
- Reference specific files and line numbers in answers
- Support follow-up questions with conversation memory
