# Chat with PDF

Intelligent conversation with any PDF document using RAG.

## Overview
Natural language conversation with PDF documents using RAG.

## Instructions
- Accept PDF file upload or file path
- Parse and chunk PDF into semantic sections
- Embed chunks using text embedding model
- Store in vector index for retrieval
- Answer questions in natural language
- Cite specific page numbers in all responses
