# Chat with YouTube Videos

Extract and chat with YouTube video transcript content.

## Overview

## Instructions

- Download transcript with timestamps
- Chunk transcript into semantic segments
- Embed and index chunks
- Answer questions about video content
- Include timestamp citations in all responses
