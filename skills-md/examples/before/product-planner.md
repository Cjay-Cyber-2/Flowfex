# Product Planner

Product Planner helps break a feature request into a practical delivery plan.

## Table of Contents

- Overview
- Workflow
- License

## Use When

- A feature request needs a clear implementation plan.
- Stakeholders need scope, risks, and milestones in one place.

## Workflow

1. Restate the feature goal in concrete terms.
2. Identify user value, constraints, and unknowns.
3. Break work into phases with owners and dependencies.
4. Call out risks, edge cases, and validation steps.

## Output Format

- Summary
- Delivery phases
- Risks
- Open questions
