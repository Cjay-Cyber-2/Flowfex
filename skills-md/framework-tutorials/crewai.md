# CrewAI Crash Course

Build collaborative multi-agent teams using CrewAI.

## Overview
Practical guide to multi-agent teams with CrewAI.

## Core Concepts
- Agent: Specialist with role, goal, and backstory
- Task: Specific work assigned to an agent
- Crew: Team of agents sharing a goal
- Process: Sequential or hierarchical execution

## Instructions
- pip install crewai crewai-tools
- Define specialist agents with clear roles
- Create tasks with expected_output specs
- Wire agents and tasks into a Crew
- Run crew.kickoff() with input variables
