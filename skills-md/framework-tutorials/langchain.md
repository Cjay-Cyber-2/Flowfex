# LangChain Crash Course

Learn LangChain fundamentals: chains, agents, memory, tools, and retrieval.

## Overview
Comprehensive crash course covering LangChain's core abstractions.

## Key Concepts
- Chains: Sequences of LLM calls and transformations
- Agents: LLMs that decide which tools to use
- Memory: Persistence across conversation turns
- Retrieval: RAG integrations with vector stores
- Tools: Callable functions agents can invoke

## Instructions
- Install: pip install langchain langchain-openai
- Build a simple prompt template and LLM chain
- Add ConversationBufferMemory for context
- Create a ReAct agent with custom tools
- Integrate RAG with FAISS or Chroma
