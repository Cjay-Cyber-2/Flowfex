# Phidata Crash Course

Build production AI agents with memory, knowledge, and tools using Phidata.

## Overview
Hands-on guide to production AI agents using Phidata.

## Core Concepts
- Agent: Primary AI actor with instructions and tools
- Knowledge: Integrated vector database for RAG
- Storage: Persistent memory backends
- Tools: Built-in and custom callable functions

## Instructions
- pip install phidata openai
- Define Agent with model, instructions, tool list
- Add PDFUrlKnowledgeBase for RAG
- Configure SqliteStorage for history
- Run and deploy with agent.serve()
