# AI 3D Pygame Agent

Build and play 3D games using Pygame with AI game logic.

## Overview
Builds 3D game environments in Pygame using raycasting.

## Instructions
- Design 3D world with raycasting perspective projection
- Define entities: player, enemies, obstacles
- Implement BFS or A* for enemy pathing
- Handle input and collision detection
- Produce runnable Pygame script with game loop
