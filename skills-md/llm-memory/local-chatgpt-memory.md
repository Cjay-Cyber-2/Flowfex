# Local ChatGPT Clone with Memory

A privacy-first local ChatGPT clone with persistent conversation memory.

## Overview
Fully local ChatGPT-like application with persistent memory, running on-device.

## Instructions
- Run local LLM via Ollama (Llama, Mistral, or Gemma)
- Store all conversations in local SQLite database
- Build searchable conversation history interface
- Implement memory summarization for long conversations
- Add user preference profiles persisted locally
- Provide web UI via Streamlit or Gradio
