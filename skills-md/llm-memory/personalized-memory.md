# LLM App with Personalized Memory

AI apps that adapt to individual users through persistent personalized memory.

## Overview
Framework for building AI apps with rich persistent user profiles.

## Instructions
- Initialize user memory profile on first interaction
- Extract preferences and expertise from conversations
- Store structured user facts: role, goals, constraints
- Retrieve relevant memory context each session start
- Adapt tone and depth based on user profile
- Support explicit memory commands: 'Remember that...'
