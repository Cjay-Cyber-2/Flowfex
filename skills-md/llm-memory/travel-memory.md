# AI Travel Agent with Memory

Travel planning with persistent memory of preferences and past trips.

## Overview
Travel planning agent with long-term memory of preferences and history.

## Instructions
- Load user travel profile from persistent memory
- Recall seat, hotel, and dietary preferences
- Reference past trips to avoid duplicates unless requested
- Continue itinerary planning from prior sessions
- Update memory with new bookings and preferences
- Produce personalized recommendations from travel history
