# Headroom Context Optimization

Reduce LLM context window usage 50-90% through intelligent summarization.

## Overview
Reduces context window usage via intelligent conversation summarization.

## Instructions
- Monitor context window usage throughout conversation
- When usage exceeds 70%, trigger summarization
- Preserve: key decisions, facts, current task state
- Compress older turns into structured summary
- Replace verbose history with summary + recent turns
- Track information retention completeness score
