# Browser MCP Agent

Control a web browser via MCP for navigation and data extraction.

## Overview
Uses Model Context Protocol to control a real web browser.

## Instructions
- Connect to browser via MCP browser tool
- Accept commands: navigate, click, type, scroll
- Extract page content and structured data
- Handle authentication flows
- Run multi-step browser workflows
- Screenshot pages for visual verification
