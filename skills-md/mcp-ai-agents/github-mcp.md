# GitHub MCP Agent

Interact with GitHub repositories via MCP.

## Overview
Connects to GitHub via MCP to read, write, and manage repository resources.

## Instructions
- Connect to GitHub MCP server
- Read files, directory structures, and commit history
- Create, update, and merge pull requests
- Manage issues, labels, and milestones
- Generate PR descriptions from diff analysis
