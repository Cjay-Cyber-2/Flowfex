# Multi-MCP Agent Router

Intelligently route tasks across multiple MCP servers.

## Overview
Intelligent dispatcher across a pool of registered MCP servers.

## Instructions
- Maintain registry of available MCP servers and capabilities
- Classify incoming tasks by required tool type
- Route to most appropriate MCP server
- Handle multi-step tasks spanning multiple MCP servers
- Aggregate results from multiple tool calls
- Report routing decisions with confidence scores
