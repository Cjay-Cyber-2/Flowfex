# AI Travel Planner MCP Agent

Plan travel using real-time data via MCP-connected travel APIs.

## Overview

## Instructions
- Connect to travel data MCP servers (flights, hotels, activities)
- Query live availability and pricing via MCP tools
- Build conflict-free itinerary from live data
- Handle booking confirmations via MCP actions
- Output complete trip plan with confirmed reservations
