# Multimodal Design Agent Team

AI design team for UI/UX, branding, illustration, and design system creation.

## Overview
Multimodal design agents for end-to-end product and brand design.

## Agents
- UX Researcher: User flows and usability analysis
- UI Designer: Component and screen design specifications
- Brand Designer: Color, typography, and brand guidelines
- Design Systems Agent: Token definitions and component library

## Instructions
- Accept product brief or design problem
- UX Researcher defines user journeys
- UI and Brand agents work in parallel
- Design Systems agent documents reusable tokens and components
