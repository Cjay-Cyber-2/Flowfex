# AI Game Design Agent Team

Collaborative game design covering mechanics, narrative, art direction, and economy.

## Overview
Multi-agent creative team for complete game design documentation.

## Agents
- Mechanics Designer: Core gameplay loops and rules
- Narrative Designer: Story, world, and character arcs
- Art Director: Visual style and asset specifications
- Economy Designer: In-game economy and monetization

## Instructions
- Accept game concept or genre as starting point
- Each agent develops their domain independently
- Synthesize into a complete game design document (GDD)
