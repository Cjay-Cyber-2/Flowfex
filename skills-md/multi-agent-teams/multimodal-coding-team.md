# Multimodal Coding Agent Team

Multi-agent coding team covering architecture, implementation, testing, and review.

## Overview
A full software development team implemented as specialized coding agents.

## Agents
- Architect: System design and component decomposition
- Implementer: Writes production code in target language
- Tester: Generates unit and integration tests
- Reviewer: Code quality, security, and style review

## Instructions
- Accept feature requirement or bug description
- Architect produces design spec
- Implementer writes code per spec
- Tester generates test suite
- Reviewer provides final quality assessment
