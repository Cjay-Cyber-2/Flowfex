# AI Teaching Agent Team

Personalized curriculum, lessons, and assessment via multi-agent education.

## Overview
Educational multi-agent system for personalized learning.

## Agents
- Curriculum Designer: Structured learning paths
- Tutor Agent: Interactive lesson delivery
- Assessor Agent: Quizzes and targeted feedback

## Instructions
- Collect learner profile: topic, level, style
- Generate structured module plan
- Deliver lessons interactively
- Adapt plan based on assessment results
