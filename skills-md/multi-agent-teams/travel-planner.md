# AI Travel Planner Agent Team

Multi-agent travel planning coordinating flights, hotels, and logistics.

## Overview
Coordinated travel specialist agents for comprehensive trip planning.

## Agents
- Flight Agent: Optimal routes and pricing
- Hotel Agent: Accommodations matching preferences
- Activity Agent: Local experiences and restaurants
- Logistics Agent: Visas, transport, currency

## Instructions
- Gather trip parameters in parallel per agent
- Validate all logistics for conflicts
- Output complete day-by-day itinerary
