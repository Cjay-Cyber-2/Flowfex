# Autonomous RAG

Self-directed RAG agent that independently plans and executes retrieval.

## Overview
Autonomous RAG that plans its own retrieval strategy without human guidance.

## Instructions
- Analyze question to identify knowledge domains needed
- Autonomously select retrieval strategies per domain
- Execute planned retrievals in parallel where possible
- Self-evaluate result quality and trigger re-retrieval if needed
- Produce final answer with full retrieval trace
