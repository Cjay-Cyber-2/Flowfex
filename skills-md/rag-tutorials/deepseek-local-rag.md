# Deepseek Local RAG Agent

Run a full RAG pipeline locally using DeepSeek models.

## Overview
Privacy-preserving RAG pipeline running entirely locally using DeepSeek.

## Instructions
- Load DeepSeek model locally via Ollama or llama.cpp
- Embed documents using local embedding model
- Store embeddings in local ChromaDB or FAISS
- Accept and embed queries locally
- Generate answers using local DeepSeek LLM
- No data leaves the machine
