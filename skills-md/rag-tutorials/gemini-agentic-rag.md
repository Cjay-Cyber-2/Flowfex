# Gemini Agentic RAG

Agentic RAG powered by Gemini with tool use for retrieval.

## Overview
Agentic RAG using Gemini's native tool calling for retrieval.

## Instructions
- Register retrieval as a Gemini function tool
- Agent calls retrieval tool when knowledge is needed
- Combine retrieved context with Gemini generation
- Support multi-hop retrieval via sequential tool calls
- Stream responses with source attribution
