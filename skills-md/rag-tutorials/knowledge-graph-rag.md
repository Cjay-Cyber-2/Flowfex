# Knowledge Graph RAG with Citations

Multi-hop reasoning over a structured knowledge graph with full citations.

## Overview
Knowledge graph construction and traversal for multi-hop Q&A.

## Instructions
- Extract entity-relation triples from source documents
- Build typed knowledge graph
- Identify seed entities from query via NER
- Traverse graph following relevant relation types
- Collect multi-hop evidence paths
- Generate answers with traced citation paths
