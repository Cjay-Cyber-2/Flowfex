# Llama 3.1 Local RAG

Privacy-first local RAG using Meta Llama 3.1.

## Overview
Fully local RAG pipeline using Llama 3.1 for privacy-sensitive workloads.

## Instructions
- Run Llama 3.1 locally via Ollama
- Embed documents with nomic-embed-text locally
- Store in local FAISS or ChromaDB index
- Process all queries and generation on local hardware
- Configure context window for optimal chunk sizes
