# Vision RAG

Multimodal RAG that retrieves and understands images and text together.

## Overview
Indexes and retrieves images alongside text for visual question answering.

## Instructions
- Index text chunks and images in shared embedding space
- Accept queries referencing visual content
- Retrieve relevant text and images jointly
- Pass to multimodal LLM for unified answer
- Cite page and image references in all responses
