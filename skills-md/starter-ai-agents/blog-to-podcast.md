# AI Blog to Podcast Agent

Convert blog posts into podcast scripts using AI.

## Overview
Converts written blog articles into engaging podcast-style scripts.

## Instructions
- Accept a blog URL or raw text
- Extract and clean main content
- Rewrite into conversational spoken format
- Add intro, transitions, and outro
- Output complete podcast script with speaker cues

## When to Use
Repurpose written content for audio audiences.
