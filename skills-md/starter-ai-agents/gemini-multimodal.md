# Gemini Multimodal Agent

Process text, images, audio, and video using Gemini.

## Overview
Processes mixed inputs using Google Gemini multimodal capabilities.

## Instructions
- Accept text, image, audio, or video inputs
- Analyze each modality
- Cross-reference findings across modalities
- Generate unified responses across input types
- Support follow-up questions about any input
