# AI Meme Generator Agent

Create contextually relevant memes from user prompts.

## Overview
Generates meme concepts with captions and image descriptions.

## Instructions
- Accept topic or situation
- Select appropriate meme format
- Generate top and bottom captions
- Describe ideal background image
- Output format name, captions, image description
