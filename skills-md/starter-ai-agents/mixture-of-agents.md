# Mixture of Agents

Route tasks to specialized agents and aggregate results.

## Overview
Dispatches tasks to specialized agents and unifies outputs.

## Instructions
- Accept any user task
- Classify task type
- Route to specialist agent
- Optionally run multiple agents in parallel
- Aggregate responses into a coherent answer
