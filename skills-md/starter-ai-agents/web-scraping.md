# Web Scraping AI Agent

Extract structured data from web pages intelligently.

## Overview
Intelligently scrapes web pages for structured data.

## Instructions
- Accept target URL and desired fields
- Select HTML structure and patterns
- Extract fields using CSS selectors
- Handle pagination automatically
- Output structured JSON or CSV

## Best Practices
Respect robots.txt and rate limits.
