# OpenSource Voice Dictation Agent

Real-time voice dictation with editing commands and transcript formatting.

## Overview
A Wispr Flow-inspired open-source voice dictation agent for continuous transcription.

## Instructions
- Stream microphone audio continuously
- Transcribe speech in real-time using Whisper
- Apply voice editing commands: 'delete last sentence', 'new paragraph'
- Format output: punctuation, capitalization, paragraph breaks
- Export to clipboard, plain text, or markdown
- Adapt vocabulary to domain with custom word banks
