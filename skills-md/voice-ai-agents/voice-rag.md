# Voice RAG Agent

Answer spoken questions using RAG over a knowledge base.

## Overview
Responds to voice queries with knowledge base retrieval and spoken answers.

## Instructions
- Convert speech input to text (STT)
- Embed query and perform semantic search
- Retrieve top-k relevant document chunks
- Generate concise natural-sounding answer
- Convert response to speech (TTS)
- Handle follow-up questions with conversation context
